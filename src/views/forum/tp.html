    常見問題

    Q1.我已購票成功，請問何時可以收到電子票券？
    本論壇採用「QR code電子票券」報到入場，將於 2020.4.6（一）起，根據您購票時所提供的聯絡資訊，陸續寄送報到簡訊及 E-mail 給您。QR code 即為您參與論壇的電子票券，您可使用手機、平板或列印紙本的方式出示 QR code 報到入場，現場無劃位，自由入座。報名時請務必填妥 E-mail 和手機等資料，以利接收行前相關通知訊息。
    Q2.我是康健的長期訂戶，是否還有優惠？
    康健雜誌為感謝訂戶的長期支持，此次癌症論壇特別提供訂戶專屬優惠，將會發送E-mail及簡訊給您，席次有限敬請把握。如有其他問題，請撥打客服專線 (02)2662-0332 轉 9。（週一至週五 9:00-17:30）
    Q3.我有團體購票需求，請問是否有購票優惠？
    癌症論壇票券購買10張以上另享優惠票價，請您撥打 (02)2507-8627分機125 曾小姐，將由專人為您服務。
    Q4.購票者和參與論壇者，可以是不同人嗎？
    可以，但請務必轉寄「QR code電子票券」給參與論壇者以供當天報到，現場無法補發電子票券。
